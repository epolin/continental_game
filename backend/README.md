# Backend del Torneo Continental

Este es el backend del proyecto "Torneo Continental". Proporciona endpoints para registrar jugadores, enviar puntajes y gestionar fichas.

## Endpoints

- `POST /api/register`: Registra un jugador.
- `GET /api/players`: Obtiene la lista de jugadores.
- `POST /api/scores`: Guarda los puntajes de los jugadores.
- `POST /api/chips`: Guarda las fichas compradas por los jugadores.
